SSD

NuDB es un almacén de valor-clave de solo adición que está optimizado para unidades SSD.

NuDB tiene un rendimiento y una huella de memoria casi constantes, independientemente de la cantidad de datos que se almacenen. NuDB requiere una unidad de estado sólido, pero usa mucha menos RAM que RocksDB para acceder a una gran base de datos.

Los servidores de producción que no son validadores deben configurarse para usar NuDB y almacenar la cantidad de datos históricos necesarios para el caso de uso.
