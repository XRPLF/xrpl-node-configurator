Servidor de API dedicado

Un servidor de API expone una API pública, no es un validador.

Escoge este modo si...

Utilizas las APIs del XRPL de manera intensiva en tu aplicación, de manera que puedes tener uno o varios servidores dedicados a este efecto. Puede que estes recogiendo estadísticas o enviando grandes volumenes de transacciones, O incluso puede que quieras exponer estas APIs para incrementar tu capital social. Si eres un negocio de tamaño medio puede que quieras correr uno de estos servidores como un respaldo de tu servidor Multifunción para que tus funciones críticas de negocio puedan continuar ante eventos de fallos de hardware. Este tipo de servidor almacena una cantidad media/alta de historial pero no persiste una gran cantidad de datos, de manera que es posible levantar clones de manera fácil o automática.
