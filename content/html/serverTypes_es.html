Tipos de Servidor

El software del XRP Ledger puede ser configurado de diferentes maneras dependiendo de las necesidades, entra otras:

  • Servidor Multifuncion
  • Validador Dedicado
  • Servidor Hub
  • Servidor con historia complete
  • Servidor API dedicado
  • Server para desarrollo

Tambien puedes utilizar el ejecutable del XRP Ledger como aplicacion de cliente para acceder las APIs de manera local.

Para mas información sobre los comandos que pueden ser ejecutados en cada modo, visita la sección de (En Inglés) Commandline Reference .
