Servidor para Desarrollo

Elige esta opción si...

Estás experimentando con el XRP Ledger, desarrollando o integrando algún tipo de aplicación que lo utiliza. No necesitas tener una calidad y estabilidad como para un sistema productivo y es posible que necesites reiniciarlo cada poco tiempo.
Necesitarás almacenar un poco de historia para poder consultar transacciones pero no necesitas tampoco tener una gran cantidad de ella.
También puede que cambies de red a menudo (Devnet, Testnet, etc...) o tengas que probar funcionalidades experimentales. Querrás tener acceso al API admin y es probable que necesites unos niveles de log mayores.
